How to reserve virtual machine (VM):

  • Set the start and end date of the reservation.
  • Set the start and end time of the reservation.
  • Choose if you want to set the VM at lock state.

You will find all your created VM reservations in "All my reservations" section. Your currently running reservations are displayed in "My virtual Machine" page. Reservation will end automatically or you can unreserve it anytime. Also you can change VM's lock state anytime and it will be unlocked at the end of your reservation.
