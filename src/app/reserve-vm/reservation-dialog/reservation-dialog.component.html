How to reserve virtual machine (VM):

  • Set the date of the reservation end.
  • Set the time of the reservation end.
  • Choose if you want to set the VM at lock state.

ou will find your reserved VM in "My virtual machines" page. Reservation will end automatically or you can unreserve it anytime. Also you can change VM's lock state anytime and it will be unlocked at the end of your reservation.
